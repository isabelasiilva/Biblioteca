<h3>Sistema de gerenciamento de biblioteca</h3>
<br>
<p>
    Este projeto implementa um sistema básico de gerenciamento de biblioteca em Java com as seguintes funcionalidades:
</p>
<ul>
    <li>Criar Autores e Livros: feita programaticamente no código-fonte (no `Main.java`).</li>
    <li>Listar Livros Disponíveis: exibe os livros que estão marcados como disponíveis.</li>
    <li>Listar Autores: exibe a lista de autores cadastrados.</li>
    <li>Listar Livros Emprestados: exibe os livros que estão marcados como indisponíveis</li>
</ul>
